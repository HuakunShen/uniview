# @uniview/example-solid-plugin-api

The **SolidJS** library of component primitives for Uniview demo plugins — the
Solid counterpart of `@uniview/example-plugin-api`. Solid plugins compose these
components, which render through `@uniview/solid-renderer` to the shared UINode
contract, so they work in any Uniview host.

Imported by the Solid demo plugin (`examples/plugin-solid-example`), not run on
its own.

```bash
pnpm build
```
