# @uniview/tui-demo

A standalone demo of the original **`@uniview/tui-renderer`** proof-of-concept —
a counter + text input using its `Box` / `Text` / `Button` / `Input` components.

> **Legacy.** `@uniview/tui-renderer` is the early POC that the test-driven
> `@uniview/tui-core` + `@uniview/tui-react` stack replaces (it had the modern
> plan's P0 issues: full-screen redraw, Unicode-width mismatch, direct
> callbacks). For new work use `examples/tui-react-demo` /
> `examples/tui-opencode-demo`. This example is kept for comparison.

```bash
pnpm --filter @uniview/tui-demo dev
```
