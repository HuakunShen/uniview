## Task 3 — AI: value function + expectimax + Node model loader

**Files:** `src/vendor/{patterns,universal,model,expectimax}.ts` (fetched), `src/ai/loader.ts` (NEW — ours), `tests/ai.test.ts`.

`src/ai/loader.ts` — replaces the upstream Web-Worker/`fetch` shell with Node `fs`, mirroring ufo's `play.test.ts`:

```ts
import { readFileSync, existsSync } from "node:fs";
import { buildValue, type Manifest } from "../vendor/model";
import type { UniversalValue } from "../vendor/universal";

/** Resolve the model dir: $UNIVIEW_2048_MODEL_DIR, else `<example>/model/`. */
export function modelDir(): string { /* env ?? default */ }
export function modelAvailable(dir = modelDir()): boolean {
  return existsSync(`${dir}/manifest.json`);
}
export function loadValue(dir = modelDir()): UniversalValue {
  const manifest = JSON.parse(readFileSync(`${dir}/manifest.json`, "utf8")) as Manifest;
  const buffers = manifest.patterns.map((_, k) =>
    manifest.parts[k].map((_p, p) => {
      const raw = readFileSync(`${dir}/lut${k}_${p}.bin`);
      return raw.buffer.slice(raw.byteOffset, raw.byteOffset + raw.byteLength);
    }),
  );
  return buildValue(manifest, buffers);
}
```

Wiring: `const ai = new Expectimax(engine, (b) => value.value(b, engine.H, engine.W))`; `ai.getMove(board, cfg: DepthCfg) → { dir, value }`.

- [ ] **RED/GREEN:** `tests/ai.test.ts` — gate the model-dependent tests behind `modelAvailable()` (`it.skipIf(!modelAvailable())`) so the suite stays green without the 85 MB asset. When present, assert: the loader builds a `UniversalValue`; `value.value(board, 4, 4)` is finite; `ai.getMove(initBoard())` returns a legal `dir`; and a short self-play loop (e.g. 200 moves, depth 1) reaches a max tile ≥ 256 (a real "the AI actually plays" check, not a smoke test).
