# Uniview Slidev

An 18-slide narrated technical explainer for Uniview: one React or Solid
component tree rendered through `UINode` to DOM, terminal, and native AppKit
hosts.

## Commands

```bash
pnpm install
pnpm dev
pnpm test
pnpm build
pnpm export:png
```

The presentation uses English on screen and in the presenter notes. Source
claims and the real TUI SVG frames come from `references/uniview` at commit
`8c74c9f`.

The deck is designed for a 16:9 screen recording. Export with click states
before visual QA because several diagrams and code examples teach through
progressive reveals.
